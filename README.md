**岗位职责：**

**职位要求：**
1. 计算机及相关专业，具备扎实的前端基础，精通HTML、CSS、JavaScript
2. 具备React.js/Vue.js等前端框架开发经验
3. 熟悉并掌握小程序开发
4. 了解数据结构、数据库等知识
5. 良好的英文阅读与文档编写能力
6. 有良好的代码习惯，结构清晰，逻辑性强，较强的学习能力
7. 能提供作品、案例等评估资料优先
